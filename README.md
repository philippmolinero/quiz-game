Test your knowledge with this trivia quiz app built with React and powered by the Trivia DB API!

## Quick Start

1. Clone the repository:

```
git clone https://github.com/your-username/quiz-game.git
```

2. Install dependencies:

```
npm install
```

3. Run the app:

```
npm start
```

4. Open the app in your browser at `http://localhost:3000`

## Usage

To play the game, simply click the "Start Quiz" button. You'll then be presented with a series of multiple-choice questions. Choose the correct answer and move on to the next question!

## Features

- Randomized questions from the Trivia DB API
- Keep track of your score and progress through the quiz
- Play again and try to beat your high score!
